# Vehicle Sales Analysis

## Descripción del Proyecto
Este proyecto consiste en el análisis de una base de datos en formato CSV sobre ventas de vehículos.  
El objetivo es explorar el comportamiento de las ventas según compañía, color, tipo de transmisión y modelo de carrocería, utilizando Power BI para la visualización de datos.
La base de datos corresponde a un dataset de práctica con fines educativos.

## Herramientas Utilizadas
- Power BI
- Microsoft Excel (limpieza preliminar)

## Indicadores Principales
- Total de registros analizados: 24.000
- Suma total de ventas: $672 millones
- Clasificación por compañía
- Distribución por colores más vendidos
- Modelos de carrocería más comunes
- Tipos de transmisión

## Vistas del Dashboard
- **Resumen general de ventas**
- **Comparativa por categorías (compañía, color, transmisión y carrocería)**

## Estructura del Repositorio
- `data/`: archivo CSV con los datos originales
- `images/`: vista previa del dashboard
- `Car Sales Analysis.pbix`: archivo Power BI  

## Objetivo del Proyecto
Demostrar habilidades en análisis de datos, manejo de datasets de tamaño medio y creación de dashboards interactivos con Power BI.
